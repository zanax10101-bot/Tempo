### Component Breakdown

**Header Bar**
- Search input with keyboard shortcut hint
- Date context (current day)

**Task Sections**
- Overdue (if any) - highlighted with error styling
- Due Today - main focus
- Quick-add at bottom of active tasks
- Completed section - collapsible

**Floating Action**
- "Start Focus" button fixed to bottom-right
- Quick access to timer from any screen

### Interactions

| Element | Action | Result |
|---------|--------|--------|
| Task row | Click | Opens task detail modal |
| Task checkbox | Click | Marks complete with animation |
| Timer icon (â±) | Click | Starts timer linked to this task |
| Priority dot | Click | Quick-change priority popup |
| Search | Focus | Opens command palette |
| + Add task | Click/Enter | Inline quick-add expands |
| Start Focus | Click | Opens full timer view |
